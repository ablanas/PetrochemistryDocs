## 2. 未定義物質の3段階分類

### 2.1 必須実装

主要チェーン、触媒製造、既存レシピ成立に必要なもの。

| 物質 | 理由 | 推奨扱い |
|---|---|---|
| HNO3 / 硝酸 | 硝酸ニッケル、硝酸コバルト、王水に必須 | 化学品蒸留塔または簡易化学合成で流体実装 |
| HCl / 塩酸 | 王水、シリカゾル、塩素化前駆体に必須 | 流体実装 |
| 王水 | 白金族鉱石から塩化白金酸を作る主工程に必須 | HNO3 + HCl の混合流体またはレシピ上の仮想試薬 |
| NH3水 / アンモニア水 | Mo/Re浸出、NH4系塩、ゼオライト交換に必須 | 流体実装 |
| NH4Cl / 塩化アンモニウム | USY化、メタバナジン酸アンモニウム晶析に必須 | アイテムまたは溶液として実装 |
| NaOH水溶液 | バイヤー法、珪酸Na、バナジウム浸出に必須 | 流体実装 |
| 珪酸Na溶液 | Na-Yゼオライトとシリカゾルに必須 | 中間流体として実装 |
| 硫酸Al | Na-Yゼオライトに必須 | アイテムまたは溶液として実装 |
| Al2O3ゾル | FCC触媒造粒で使用 | シリカゾルと同格の触媒バインダーとして実装 |
| 天然ガスまたはSMR用ガス | SMR燃料として記載あり | LPGへ簡略化するなら定義不要。天然ガスを残すなら流体実装 |
| 塩素ガス / Cl2 | 改質触媒塩素化、EDC/VCM系で使用 | 触媒再活性化を実装するなら必須 |

### 2.2 簡略化可能

リアルさは上がるが、ゲーム上は既存物質や抽象試薬へまとめられるもの。

| 物質 | 簡略化案 | 理由 |
|---|---|---|
| 水ガラス | 珪酸Na溶液へ統合 | 用途が近い |
| CaO | 石灰石粉または酸化マグネシウムで代替 | 担体成形補助として抽象化可能 |
| 無水MgCl2 | ZN触媒前駆体へ統合 | ZN触媒を細分化しすぎないため |
| TiCl4 | ZN触媒前駆体へ統合 | 高度化学品として直接入手扱いでもよい |
| エタノール | 溶媒タグへ統合 | 単体実装しなくても触媒工程は成立する |
| ヘキサン | 溶媒タグへ統合 | 重合溶媒として抽象化可能 |
| DMSO | 硫化剤アイテムへ統合 | HDS触媒硫化剤のフレーバーでよい |
| TEA | ZN助触媒へ統合 | 個別実装しなくても重合炉の触媒消費で表現可能 |
| ジイソブチルフタレート | 電子供与体へ統合 | ZN触媒の過度な細分化を防ぐ |
| 外部電子供与体シラン | 電子供与体へ統合 | 同上 |
| ホスゲン | TDI前駆体として抽象化 | 高度・危険化学品のため将来まで抽象化推奨 |
| Sb2O3 | PET触媒タグへ統合 | PET重合の触媒名として表示だけでもよい |

## 3. Steam Cracker内部設備の整理

Steam Crackerはプレイヤーが建設する主要設備として維持する。ただし内部の急冷・分離をどの粒度で扱うかを明確にする必要がある。

### 3.1 推奨分類

| 要素 | 分類 | 理由 | 推奨実装 |
|---|---|---|---|
| スチームクラッカー本体 | 独立設備 | プレイヤーが建設する主要プロセス設備 | 7x12x9マルチブロック |
| 急冷水塔 | 補助設備 | 未接続で収率ペナルティがあり、機能的には本体外の補助ユニット | Steam Crackerに隣接する2x?x?補助マルチブロック、または本体拡張区画 |
| 極低温分離器 | 補助設備 | 分解生成物の分離に重要で、RF大量消費という独自機能がある | Steam Crackerに隣接する補助マルチブロック |
| TLE | 内部部品 | 分解炉出口に直結する熱回収部品で、独立操作設備にしない方がよい | `heat_recovery_block` の高温急冷タグ |
| ラジアントバーナー | 内部部品 | 炉内加熱部品 | `burner_block` |
| クラッキングコイル | 内部部品 | 炉管そのもの | `heat_exchange_tube` または専用 `cracking_coil` 表示 |
| 急冷水ノズル/冷却水ポート | 内部部品 | 急冷水塔の入出力部 | `metering_port` |
| 低温冷却器/分離ドラム | 内部部品 | 極低温分離器の内部構成 | `condenser_block` / `heat_exchange_tube` |

### 3.2 判断

- 急冷水塔と極低温分離器は「内部部品」ではなく「補助設備」として扱うのがよい。
- 理由は、ロードマップに `QuenchTowerBlockEntity` と `CryoSeparatorBlockEntity` が記載され、未接続ペナルティやRF消費など独自の挙動があるため。
- ただし、CDUやFCCのような主要プロセス設備にはしない。Steam Cracker専用の補助設備として扱う。

## 5. 修正優先度

### 最優先

- ポート類を4種類へ統合する。
- バーナー類を2種類へ統合する。
- Steam Crackerの急冷水塔と極低温分離器を補助設備として明記する。
- 未定義物質のうち必須実装を2.6または2.7へ追加する。

### 次点

- 熱交換器部品を5種類へ統合する。
- 5章の構成表から詳細レシピを段階的に外し、部品名と必要数中心にする。
- 詳細構成不足のBTX抽出装置、クラウス装置、高圧重合炉、縮合重合反応器、調合装置を補完する。

### 後回し

- ZN触媒の溶媒・電子供与体の細分化。
- 二次化学品反応器群の詳細設備化。
- ポリマー派生品の全モノマー製造チェーン。

## 6. 結論

- 共通部品は現状かなり細かく、バーナー、ポート、熱交換器を中心に統合余地が大きい。
- 設備そのものは統合せず、プレイヤーが建設する設備として維持する方針が妥当。
- Steam Crackerは本体を独立設備、急冷水塔と極低温分離器を補助設備、TLEやコイルを内部部品として整理するのが最も整合的。
- 未定義物質はすべて同時に実装せず、主要チェーン成立に必要な酸・塩基・アンモニア・塩素系から実装するのがよい。
