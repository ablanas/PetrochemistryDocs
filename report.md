# 化学工学プラントMOD 再整理レポート

## 共通化した部品

- 塔内部品: 蒸留トレイ、化学品蒸留トレイ、構造化充填材、液体分散板、支持グリッド、交換樹脂充填ブロック、リフラックスドラム
- 熱交換部品: ステンレス管バンドル、インコネル管、電熱リボイラー、空冷コンデンサー、管式コンデンサー、TLE、冷却ジャケット
- 反応器部品: 触媒充填層、分散プレート、サイクロン分離器、触媒マンウェイ、触媒ホッパー、触媒計量注入口
- 加熱炉部品: 重油バーナー、ラジアントバーナー、中間加熱バーナー、炉内コイル支持架、高温排ガス煙突
- 配管・流体系: 精密バルブ、フランジ継手、流体ポート、蒸気ヘッダー接続口、ガス導入ポート

## 共通化候補

- サイクロン分離器とサイクロン集塵は同一レシピのため、同一部品IDまたは派生テクスチャ扱いにできる。
- ラジアントバーナー、天井バーナー、中間加熱バーナー、対向バーナーは同系統レシピのため、バーナー種別タグで整理できる。
- 空冷コンデンサー、管式コンデンサー、TLE、熱回収交換器は熱交換器チューブを共通中間部品として参照できる。
- 蒸留トレイと化学品蒸留トレイは用途差を残しつつ、トレイ系タグで扱える。
- 流体ポート、ガス導入ポート、スラリー排出ポート、電解液循環ポート、廃液排出弁は配管・ポート系タグで扱える。

## 未定義物質

- HNO3 / 硝酸: レシピで使用されるが、流体または化学品IDとして未定義。
- HCl / 塩酸: 王水、塩素化工程で使用されるが、流体または化学品IDとして未定義。
- 王水: 白金族鉱石処理で使用されるが、流体IDと製造レシピが未定義。
- アンモニア水 / NH3 / NH4Cl: アンモニア浸出、NH4-Y化、メタバナジン酸アンモニウム晶析で使用されるが、入手経路が未定義。
- NaOH水溶液、珪酸Na、硫酸Al、水ガラス、CaO: 触媒担体・ゼオライト工程で使用されるが、明確な物質定義が不足。
- 無水MgCl2、TiCl4、TEA、電子供与体、ジイソブチルフタレート、外部電子供与体シラン、ヘキサン、エタノール: ZN触媒工程で使用されるが、化学品一覧または入手経路が不足。
- 塩素ガス / Cl2: 改質触媒再活性化とEDC工程で使用されるが、流体定義と製造経路が不足。
- DMSO: HDS触媒硫化剤として使用されるが、化学品定義が不足。
- 天然ガス: SMR燃料として記載されるが、流体一覧に未定義。
- Sb2O3、BPA、エピクロロヒドリン、ポリオール、ホスゲン: ポリマー・二次化学品で使用されるが、定義や製造経路が不足。
- NaCN: ロードマップに記載があるが、物質定義と製造レシピが不足。

## 孤立定義

- HCCは設計思想に存在するが、設備一覧・レシピ・ロードマップ詳細との接続が弱い。
- 高圧重合炉、縮合重合反応器、クラウス装置、BTX抽出装置、調合装置は4章にあるが、現時点では5章の詳細構成がないため参照が不完全。
- 一部の最終ポリマー（PS、PVC、PU、SBR、エポキシ樹脂）は2.8で定義されるが、上流モノマーや二次反応設備の詳細実装が未整備。

## 循環依存

- 回転焼成炉は耐熱煉瓦を必要とし、耐熱煉瓦は回転焼成炉で製造される。5.3では初回のみ炉窯ブロック代用と記載されているため、例外ルートを明文化すれば解消可能。
- EAFは耐熱煉瓦を必要とし、ステンレス鋼外壁など多くの上位部品はEAFを必要とする。耐熱煉瓦の初回製造ルートが明確であれば循環は解消可能。
- 触媒調製炉で製造するセラミックファイバーブランケットが断熱キャスタブルに使われるため、触媒調製炉自体が断熱キャスタブルを要求しない現行構成は妥当。今後、触媒調製炉に断熱キャスタブルを要求する場合は循環に注意。

## 設備統合候補

以下は提案のみであり、勝手に統合しない。

- 触媒製造ライン: 含浸ユニット、噴霧乾燥機、触媒調製炉をまとめた高コスト後期設備の候補。ただし、現行の各設備は触媒工程の段階差を表すため維持が望ましい。
- 粉体処理ライン: 破砕機、ボールミル、回転焼成炉、還元焼成炉を一連の処理ラインとして扱う候補。ただし、序盤から中盤のゲーム進行では個別設備の方が理解しやすい。
- 湿式精製ライン: 湿式浸出槽、イオン交換塔、晶析装置を統合する候補。ただし、酸浸出、イオン交換、晶析はゲーム上の役割が異なるため、統合は後期自動化オプションに留めるのがよい。
