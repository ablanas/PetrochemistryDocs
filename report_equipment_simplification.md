# 中間加工設備削減レビュー

## 前提

- このレポートは `design.md` のレビュー結果であり、`design.md` は編集しない。
- CDU、VDU、FCC、HDS装置、接触改質装置、Steam Cracker、SMR装置は主要プロセス設備として維持する。
- PSA、遠心分離機、化学品蒸留塔、電解精錬槽も独立設備として維持する。
- 目的は化学工学らしさを失わず、プレイヤーが建設する中間加工マルチブロック数とレシピカテゴリ数を減らすことである。

## 現状整理

`design.md` の4.2には中間加工機械として20設備が定義されている。

| 区分 | 設備 |
|---|---|
| 粉砕・選鉱 | 破砕機、ボールミル、浮遊選鉱機 |
| 湿式・電解 | 湿式浸出槽、電解精錬槽、イオン交換塔、晶析装置 |
| 焼成・熱処理 | 焙焼炉、回転焼成炉、還元焼成炉、触媒調製炉 |
| 触媒製造 | 含浸ユニット、噴霧乾燥機、触媒調製炉 |
| 分離・精製 | PSA、遠心分離機、化学品蒸留塔 |
| 反応・補助 | オートクレーブ反応器、硫酸製造装置 |
| 合金・序盤 | 電気アーク炉、簡易精錬炉 |

現状の問題は、触媒製造と湿式精製で「工程ごとの専用設備」が多く、プレイヤーが建設するマルチブロックが増えている点にある。

## 統合推奨

### 1. 触媒製造設備

対象:

- 含浸ユニット
- 噴霧乾燥機
- 触媒調製炉

提案:

- `触媒製造ライン` へ統合する。
- モードは `含浸モード`、`乾燥モード`、`焼成モード` とする。
- 触媒調製炉は独立設備名としては廃止候補だが、内部部品またはモード名として残す。

評価:

| 観点 | 評価 |
|---|---|
| 統合メリット | 触媒製造のために3台建てる負担を1台に減らせる。触媒工程の導入が分かりやすくなる。 |
| 統合デメリット | 噴霧乾燥と含浸は化学工学的にはかなり別設備であり、現実性は少し落ちる。 |
| ゲームプレイへの影響 | 中盤の触媒導入が軽くなる。FCC、HDS、改質、SMR触媒のクラフト導線が短くなる。 |
| レシピへの影響 | 3.3の触媒製造レシピは残し、使用機械を `触媒製造ライン: 各モード` に置き換える。レシピ数自体は大きく減らないが、レシピタイプは3設備分から1設備分へ減る。 |
| マルチブロックへの影響 | 4x6x4程度の中型ラインにし、内部に含浸ドラム、噴霧ノズル、加熱炉ブロックを配置する構成が妥当。 |
| 化学工学的妥当性 | 「触媒製造パイロットライン」と見なせば許容できる。完全な単一装置ではなく、統合ラインとして扱うのが重要。 |

判定:

- 統合推奨。
- 最も削減効果が大きく、プレイヤー負担の改善も明確。

### 2. 焼成設備

対象:

- 回転焼成炉
- 還元焼成炉

提案:

- `焼成炉` へ統合する。
- モードは `通常焼成`、`酸化焼成`、`還元焼成` とする。
- 還元焼成は水素ポート、還元チューブ、雰囲気制御を要求する上位モードにする。

評価:

| 観点 | 評価 |
|---|---|
| 統合メリット | 焼成系の設備数を減らせる。回転焼成炉を作った後に似た高温炉を再度作る負担が下がる。 |
| 統合デメリット | 回転焼成炉と還元焼成炉は形状も雰囲気も異なるため、そのまま同一装置にすると不自然。 |
| ゲームプレイへの影響 | 耐熱煉瓦、Al2O3、V2O5、触媒前駆体の熱処理が1設備に集まり、JEI上の理解が容易になる。 |
| レシピへの影響 | 回転焼成炉レシピと還元焼成炉レシピを `焼成炉` のモード違いへ統合する。還元系は水素消費を条件に残す。 |
| マルチブロックへの影響 | 基本構造は回転焼成炉寄りにし、還元モードでは還元チューブ、H2入力、排ガスポートを追加要求する。 |
| 化学工学的妥当性 | 「多目的雰囲気制御キルン」とすれば妥当。単純な回転炉そのものに還元チューブを混ぜるより、上位モードとして扱うべき。 |

判定:

- 統合推奨。
- ただし `焼成炉` は単なる小型炉ではなく、雰囲気制御可能な多目的キルンとして定義する。

### 3. 湿式精製設備

対象:

- 湿式浸出槽
- イオン交換塔
- 晶析装置

提案:

- `湿式精製ライン` へ統合可能。
- モードは `浸出`、`イオン交換`、`晶析` とする。
- ただし化学工学的には3者の設備差が大きいため、統合するなら「1ブロック装置」ではなく「ライン型マルチブロック」とする。

評価:

| 観点 | 評価 |
|---|---|
| 統合メリット | 鉱石精製から触媒塩製造までの湿式処理設備を1台にまとめられる。中間加工設備の建設数削減効果が大きい。 |
| 統合デメリット | 浸出槽、イオン交換塔、晶析槽は現実にはかなり異なる。単一槽で全部できると化学工学らしさが薄くなる。 |
| ゲームプレイへの影響 | 金属塩、ゼオライト交換、晶析の導線が簡略化される。触媒原料の製造が軽くなる。 |
| レシピへの影響 | 3.1の湿式浸出、イオン交換、晶析レシピを `湿式精製ライン` のモード違いへ統合する。レシピ数は残るが、レシピタイプは3設備分から1設備分へ減る。 |
| マルチブロックへの影響 | 5x5x5程度のライン設備にし、交換樹脂層、冷却コイル、撹拌ブロックをモード別に有効化するのが妥当。 |
| 化学工学的妥当性 | 「湿式精製ライン」としてなら許容可能。単独の万能槽として扱うのは非推奨。 |

判定:

- 条件付きで統合推奨。
- Minecraft上の負担削減を優先するなら採用価値が高い。
- 化学工学らしさを強めるなら、イオン交換塔だけは独立維持してもよい。

## 統合非推奨

### 主要プロセス設備

以下は統合しない。

- CDU
- VDU
- FCC
- HDS装置
- 接触改質装置
- Steam Cracker
- SMR装置

理由:

- それぞれ出力物、操作条件、ゲーム上の役割が大きく異なる。
- これらを単位操作へ統合すると、プロセスチェーンの読みやすさと達成感が失われる。

### 独立維持指定設備

以下も統合しない。

- PSA
- 遠心分離機
- 化学品蒸留塔
- 電解精錬槽

評価:

| 設備 | 統合非推奨理由 |
|---|---|
| PSA | ガス吸着分離という役割が明確で、SMR後の水素精製ゲートとして重要。 |
| 遠心分離機 | 固液分離の汎用設備で、ゼオライトやスラリー処理の工程識別に必要。 |
| 化学品蒸留塔 | 溶媒・酸・化学品精製に使う蒸留設備で、石油系CDU/VDUと役割が違う。 |
| 電解精錬槽 | 金属回収・電着の独自設備で、湿式精製ラインへ吸収すると電力大量消費と電極管理の意味が薄れる。 |

### その他、統合しない方がよい設備

| 設備 | 判定 | 理由 |
|---|---|---|
| 破砕機 | 非推奨 | 序盤の鉱石処理入口として分かりやすく、利用頻度も高い。 |
| ボールミル | 非推奨 | 粉砕とスラリー化を担い、破砕機と段階差がある。 |
| 浮遊選鉱機 | 非推奨 | 鉱石濃縮の役割が明確。粉砕設備へ混ぜると選鉱工程が消える。 |
| 焙焼炉 | 非推奨 | 硫化鉱の酸化とSO2回収に特化しており、焼成炉へ統合すると硫酸チェーンが曖昧になる。 |
| 硫酸製造装置 | 非推奨 | SO2からH2SO4を作る専用化学プラントであり、湿式精製や焼成炉とは役割が違う。 |
| オートクレーブ反応器 | 非推奨 | 高圧水熱合成の設備で、触媒製造ラインへ入れるとゼオライト合成の高圧性が消える。 |
| 電気アーク炉 | 非推奨 | 合金・高温精錬の基盤設備。中間加工の進行ゲートとして必要。 |
| 簡易精錬炉 | 非推奨 | 序盤救済設備であり、マルチブロック負担にはならない。 |

## 削減効果

### 設備数

現状の中間加工設備数:

- 20設備

統合後:

- 15設備

削減:

- 5設備削減
- 中間加工設備ベースで25%削減

内訳:

| 統合 | 現状 | 統合後 | 削減 |
|---|---:|---:|---:|
| 触媒製造設備 | 3 | 1 | -2 |
| 湿式精製設備 | 3 | 1 | -2 |
| 焼成設備 | 2 | 1 | -1 |
| 合計 | 8 | 3 | -5 |

### マルチブロック数

プレイヤーが建設する中間加工マルチブロックは、20種類から15種類へ減る。

効果:

- 触媒製造開始に必要な建設数が3台から1台になる。
- 湿式精製チェーンに必要な建設数が3台から1台になる。
- 焼成系の高温炉建設が2台から1台になる。

注意:

- 統合後ライン設備は少し大型化するため、総ブロックコストは単純に5台分減らさず、60-70%程度に圧縮するのがよい。
- 設備数は減らすが、内部部品として含浸ドラム、噴霧ノズル、交換樹脂、冷却コイル、水素ポートなどを残すと化学工学らしさを維持できる。

### レシピ数

レシピの「工程数」は大きく減らさない方がよい。減らすべきなのは「使用設備カテゴリ」と「JEIカテゴリ」である。

推定効果:

| 項目 | 現状 | 統合後 | 効果 |
|---|---:|---:|---|
| 対象設備レシピカテゴリ | 8カテゴリ | 3カテゴリ | -5カテゴリ |
| 対象設備のクラフトレシピ | 8設備分 | 3設備分 | -5設備分 |
| 触媒製造工程レシピ | ほぼ維持 | ほぼ維持 | モード違いへ移行 |
| 湿式精製工程レシピ | ほぼ維持 | ほぼ維持 | モード違いへ移行 |
| 焼成工程レシピ | ほぼ維持 | ほぼ維持 | 雰囲気モードへ移行 |

推奨:

- 素材・触媒・化学品のレシピは維持する。
- JEIカテゴリを `触媒製造ライン`、`湿式精製ライン`、`焼成炉` に集約する。
- レシピJSONは機械IDを統合し、`mode` または `operation_type` を追加する。

## 最終提案

### 推奨構成

Minecraftのゲームプレイを優先しつつ化学工学らしさを維持するなら、以下の構成を推奨する。

1. 触媒製造設備は `触媒製造ライン` へ統合する。
2. 回転焼成炉と還元焼成炉は `焼成炉` へ統合する。
3. 湿式浸出槽、イオン交換塔、晶析装置は `湿式精製ライン` へ統合する。ただし内部モジュールとして交換樹脂層、冷却コイル、撹拌槽を残す。
4. PSA、遠心分離機、化学品蒸留塔、電解精錬槽は独立維持する。
5. 主要プロセス設備は絶対に統合しない。

### 化学工学面での評価

- 触媒製造ラインは現実の「触媒製造プラントライン」として見れば妥当。
- 湿式精製ラインは単一槽ではなく、浸出区画、交換区画、晶析区画を持つラインとして表現すれば妥当。
- 焼成炉は多目的雰囲気制御キルンとして扱えば妥当。
- 遠心分離、PSA、化学品蒸留、電解精錬は単位操作の違いが大きいため独立維持が妥当。

### ゲームプレイ面での評価

- 中盤の設備建設負担が大きく下がる。
- 触媒製造のために小型マルチブロックを多数並べる必要がなくなる。
- プレイヤーは「粉砕」「湿式精製」「焼成」「触媒製造」「分離」「電解」という大きな工程単位で理解できる。
- 工程のリアルさはモード、消費流体、内部部品、温度/雰囲気条件で維持できる。

### 最終判定

統合案は採用推奨。

ただし、単純な万能機械へ統合するのではなく、以下の3設備を「ライン設備」または「モード付き多目的設備」として定義すること。

- 触媒製造ライン
- 湿式精製ライン
- 焼成炉

この構成なら、中間加工設備は20から15へ削減でき、化学工学らしさも大きく損なわない。
